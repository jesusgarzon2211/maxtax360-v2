# MaxTax360° — Landing Page v2

Landing page profesional para MaxTax360° Tax & Business Services.

## Stack
- HTML5 + CSS3 + Vanilla JS
- Sin frameworks — rápido y simple
- Google Fonts: Bebas Neue, DM Serif Display, Manrope, JetBrains Mono

## Estructura
```
assets/images/   → Imágenes (max.png = mascota MAX)
BRAND.md         → Sistema visual y de marca
COPY.md          → Todos los textos aprobados

index.html       → Landing page completa
```
